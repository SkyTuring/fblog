01背包的状态转移方程：

```txt
F[i,v] = max{F[i-1,v], F[i-1,v-Ci] + Wi}
```

完全背包的状态转移方程：

```txt
F[i,v] = max{F[i-1,v-kCi] + kWi | 0 <= kCi <= v}
```

多重背包的状态转移方程：

```txt
F[i,v] = max{F[i-1,v-kCi] + kWi | 0 <= k <= Mi}
```
